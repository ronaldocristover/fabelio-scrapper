
## Deployment Instruction

1. Clone the repository
2. Install Laravel Depedency
```bash
composer install
```
3. generate Laravel Key
```bash
php artisan key:generate
```

4. Copy .env-example into .env and change into you mysql database environtment

5. run migration
```bash
php artisan migrate
```
6. run the app 
```bash
{yourhost}/public/
```
